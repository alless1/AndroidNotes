c语言特性
1.只有基本数据类型,没有引用数据类型;
2.不指定函数返回值,默认返回值类型为int;
3.控制台调用命令-暂停	system("pause")	可以用来调用java程序;
4.控制台输出字符串方式和java有区别.printf("i=%d",i);	d代表后面的i的参数类型(整数);
%#X 是以十六进制方式打印;
5.变量在使用前要先声明;
6.long 占4个字节,可以用方法sizeof(long)查看,long long占8个字节;

7.&取变量的地址,*取地址上的值(不光是取值,而是指向值所在的位置),*后面跟的是地址(指针).
不带*是普通的变量值,带*号的是带地址引用的变量值.

8.指针是用来表示一块内存地址的.
  指针变量是用来存放一个内存地址的.
利用指针可以在子函数里面修改主函数里面的数据(引用传递);
问:java中有值传递和引用传递吗?
答:java中只有值传递.Person p = new Person();p里面存放的内容 就是person对象的地址.把地址复制了一份.

9.把两颗星看做一个整体符号吧，一个星的指针存放的是变量的地址。两颗星的指针存放的是指针的地址，这样理解一下。
举例：int i=5;
int *p=&i;
int **q=&p;
此时，一级指针p存放的是变量i的地址，通过p可以从地址层上操作i。
其次，二级指针q存放的是指针变量p的地址，通过q可以从地址层上去改变p的值，改变的p的值意味着什么？意味着p的指向发生了改变。

10.表示字符串的简洁方式
char* arr = "hello,world"

11.数组是一块连续的内存控件  数组名 就是内存空间的首地址.
数组名[i] == *(数组名+i);

12.在堆内存动态申请空间 int* p=(int*)malloc(sizeof(int)*10);改变大小重新申请 p = realloc(p,sizeof(int)*12);手动释放free(p);

13.结构体.struct Student; (*pst).age 或者pst->age;
14.联合体
首先，联合体的各个成员共用内存，并应该同时只能有一个成员得到这块内存的使用权（即对内存的读写），而结构体各个成员各自拥有内存，各自使用互不干涉。所以，某种意义上来说，联合体比结构体节约内存。
